# Three-Tier Web Application Deployment on AWS

## Overview

This project demonstrates the deployment of a three-tier web application on AWS using a production-style architecture.

## Architecture

- Web Tier: Nginx (EC2 - Public Subnet)
- Application Tier: Flask + Gunicorn (EC2 - Private Subnet)
- Database Tier: MySQL (EC2 - Private Subnet)

## AWS Services Used

- Amazon EC2
- Amazon VPC
- Internet Gateway
- NAT Gateway
- Route Tables
- Security Groups
- IAM
- AWS Systems Manager (SSM)

## Tech Stack

- Python
- Flask
- Gunicorn
- Nginx
- MySQL
- Linux

## Features

- View Employees
- Add Employee
- Delete Employee
